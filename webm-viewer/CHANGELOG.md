# Changelog

## [0.1.0] — 2026-04-27

- Opens `.webm` files in a custom editor panel automatically on double-click.
- HTML5 `<video>` element with play/pause, seek, volume, playback speed, and fullscreen controls.
- Keyboard shortcuts: Space/K (play/pause), ←/→ (seek ±5s), ↑/↓ (volume), M (mute), F (fullscreen), 0 (jump to start).
- Playback speed cycle: 0.25×, 0.5×, 0.75×, 1×, 1.25×, 1.5×, 2×.
- Error overlay with specific MediaError messages.
- Strict Content Security Policy with per-session nonce.
- No external CDN dependencies.
